Machine a ecrire

Ce code est fait pour un texte simple sans balisage ou légé.
Il ne peut pas contenir deux balisages consécutifs, par exemple pas de balisage d'element ouvarant et/ou fermant et/ou orphelines à la suite.
Ou encore pour les caractères spécieaux tels que
&amp; comme vous le constatez l'esperluette bug.

En cliquant sur le clavier cette foie-ci un texte va s'écrire lettre par lettre.
